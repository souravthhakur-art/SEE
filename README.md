# Palum Dhara Storefront

Palum Dhara is a premium Indian food and lifestyle brand building a modern ecommerce experience rooted in Himalayan craftsmanship, authenticity, and trust.

This repository contains the Next.js storefront and the Palum Dhara Operating System (PDOS) documentation used to guide all engineering work.

---

## Technology Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- Prisma (planned)
- Razorpay (planned)

## Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Production build:

```bash
npm run build
```
